🛠 Manutenzione Completa Bandi

Un clic = ciclo completo: riattiva bandi validi, cerca nuovi da fonti ufficiali, elimina solo veri scaduti, rimuovi duplicati certi (stesso URL o stesso titolo+data), verifica titoli e link. Automatico ogni 7 giorni.

Avvio...

Regole SKILL.md Applicate

✓ Ricerca nuovi bandi — 30+ fonti ufficiali — Interroga incentivi.gov.it (JSON + CSV), 20 portali regionali (HTML scraping), 5 enti nazionali (Invitalia, SIMEST, INAIL, SACE, OpenCoesione), 3 Ministeri (MIMIT, MUR, MASE), 2 aggregatori CCIAA (Unioncamere, ContributiRegione), Horizon Europe

✓ Parser multi-formato — JSON per API strutturate, CSV per open data, HTML scraping per portali regionali e enti senza API

✓ Anti-plagio titoli — I titoli vengono riformattati in "NomeBando — Descrizione Breve Originale"

✓ Pulizia bandi scaduti — Bandi con scadenza passata vengono disattivati automaticamente

✓ Deduplicazione avanzata — 3 fattori: stesso URL = duplicato certo, stesso titolo + stessa data = duplicato, date diverse = MAI duplicato

✓ Verifica link — URL controllati con HEAD request + rilevamento link generici (homepage, /bandi/, /index.html)

✓ Data aggiornamento — Registrata ad ogni esecuzione (manuale o automatica)
