### 3.1 ERRORI NATURALI (5-8 errori casuali, MAI forzati):

□ "prezzibloccati" (incollato senza spazio)
□ "ristrutturazzione" (doppia Z casuale)
□ "," mancante: "Infatti Aruba rincarano"
□ "202 6" (spazio nel numero)
□ "PageSpeed Garantito" (maiuscola casuale)
□ Frase troncata: "Le web agency de..."
□ Apici italiani: «vero valore»
□ Ripetizione: "crescita, la crescimento"

**ESEMPI CORRETTI:**
- "RaaS blocca i prezi per sempre" → "prezzi" scritto male naturalmente
- "Aruba parte da 9,90€poi rincarano" → virgola mancante
- "PageSpeed 95 + garantito" → spazio extra casuale

**EVITARE:** 
- ❌ "xe calda caldissima" (dialetto Veneto NON VA BENE per web agency)
- ❌ Troppi errori consecutivi (max 1 ogni 300 parole)

**REGOLA:** Nessun dialetto Veneto in articoli web agency. Tono professionale ma accessibile.
